Добиться того, чтобы не умеющие летать, не могли этого делать, а также того, чтобы не умеющие плавать, не плавали, можно с помощью Override.
Для этого нужно будет заходить в каждый класс и отдельно для него указывать Override в зависимости от того, может ли животное летать, плавать или ходить. 
Предположу, что удобнее было бы сделать 3 отдельных класса для летающих, плавающих и остальных животных. 
И от этих классов уже можно было бы сделать классы-наследники.
